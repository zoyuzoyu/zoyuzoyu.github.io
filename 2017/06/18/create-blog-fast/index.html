使用github快速创建个人博客

发表于 2017-06-18

注意

环境：macOS Sierra 10.12版本。其他环境酌情修改安装方式。

一.创建GitHub账号和仓库

1.首先你需要注册一个github账号，注意username，这会影响到你的域名，你的域名将会是 username.github.io ，所以认真的取个名字吧。

2.创建一个仓库(repository)来存储我们的网站数据，Respository name 一栏中写入上面准备好的 username.github.io。Owner需要确认为相同的username。

二.安装Hexo

4.写文章

5.测试

测试服务启动，你可以在浏览器中输入https://localhost:4000 访问了你的博客了。

6.安装hexo-deployer-git自动部署发布工具

7.发布
测试没问题后，我们就生成静态网页文件发布至我们的Github pages中。

如果这是你的第一次，终端会让你输入Github 的邮箱和密码。
正确输入后，骚等片刻，就会把你的博客上传至Github了。
以后在每次把博客写完后，执行一下这个命令就可以直接发布了。

8.完成
恭喜你，你已经创建了你的个人博客。
打开 http://username.github.io
欣赏你的作品吧!
