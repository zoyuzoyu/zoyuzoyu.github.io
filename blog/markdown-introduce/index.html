Markdown 简介

发表于 2017-06-19

Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的HTML文档。
Markdown的语法简洁明了、学习容易，而且功能比纯文本更强，因此有很多人用它写博客。

Markdown 基本语法

输出后的效果 Markdown
粗体 **text**
斜体 *text*
链接 [title](http://)
内置代码 `code`
图片 [alt](http://)
列表 * list
引用 > text
标题1 # Header 1
标题2 ## Header 2
标题3 ### Header 3
